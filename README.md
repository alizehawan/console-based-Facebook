# Console-Based Facebook Simulation

This project demonstrates **Object-Oriented Programming (OOP) concepts** in C++.

## Features
- User class with attributes like name, email, password.
- Methods for login, posting status, and adding friends.
- Encapsulation, inheritance, and basic polymorphism demonstrated.
- Console-based interaction.

## Concepts Highlighted
- Classes & Objects
- Encapsulation
- Inheritance
- Polymorphism
- Constructors & Destructors
